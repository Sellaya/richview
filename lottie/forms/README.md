# Lottie Form Assets

Place Lottie JSON files here for form feedback animations.

**Required files (optional enhancement):**
- `spinner-loading.json` — Loading spinner (~1s loop, <20KB)
- `success-check.json` — Success checkmark (play once, 500–700ms, <20KB)

**Fallback:** The site works without these. CSS/SVG fallbacks are used when:
- Files are missing
- `prefers-reduced-motion: reduce` is set
- Lottie library fails to load

**Sources:** [LottieFiles](https://lottiefiles.com) — search "minimal loader" and "success check"
